Pièces hexagonales Resident Evil 4 Remake : comment résoudre l'énigme du piédestal ?

Si vous avez commencé à jouer à ce Resident Evil 4 Remake, vous avez sûrement remarqué qu'il est bourré de petites énigmes et c'est justement de l'une d'elles dont nous allons vous parler dans cet article ! Plus précisément, c'est l'énigme du piédestal avec les hexagonales qui va nous intéresser.

s

Resident Evil 4 Remake est un jeu d'action-aventure populaire dans lequel les joueurs doivent résoudre des énigmes complexes pour progresser dans le jeu. L'une de ces énigmes est celle des pièces hexagonales sur le piédestal. Dans cette énigme, les joueurs doivent trouver quatre pièces hexagonales pour les placer sur un piédestal. Chaque pièce est de couleur différente et comporte une gravure unique. Les joueurs doivent trouver et placer les pièces dans l'ordre correct pour résoudre l'énigme.

Où trouver les pièces hexagonales ?

La première étape pour résoudre cette énigme est de trouver les quatre pièces hexagonales. Elles sont réparties dans différents endroits du jeu. Voici où les trouver : La première pièce hexagonale se trouve dans la salle des trophées du château. Pour y accéder, les joueurs doivent utiliser la clé de l'aigle et la clé de la salamandre pour ouvrir les portes verrouillées. La pièce hexagonale est située sur un tableau. La deuxième pièce hexagonale se trouve dans le laboratoire. Les joueurs doivent utiliser la clé du serpent pour accéder à cette zone. La pièce hexagonale est cachée derrière une armoire. La troisième pièce hexagonale se trouve dans la grotte. Les joueurs doivent traverser la grotte pour atteindre une zone où ils peuvent utiliser la clé de la panthère pour ouvrir une porte verrouillée. La pièce hexagonale est située sur une statue. La quatrième pièce hexagonale se trouve dans le château. Les joueurs doivent utiliser la clé de la reine pour accéder à cette zone. La pièce hexagonale est cachée dans un placard.

Comment placer les pièces hexagonales sur le piédestal ?

Une fois que les joueurs ont trouvé les quatre pièces hexagonales, ils doivent les placer dans le bon ordre sur le piédestal. Voici comment résoudre cette énigme : Placez la pièce hexagonale rouge sur la première encoche du piédestal. Placez la pièce hexagonale verte sur la deuxième encoche du piédestal. Placez la pièce hexagonale bleue sur la troisième encoche du piédestal. Enfin, placez la pièce hexagonale jaune sur la quatrième et dernière encoche du piédestal. Une fois que les joueurs ont placé les pièces hexagonales dans le bon ordre, une porte s'ouvrira et ils pourront continuer leur progression dans le jeu.

Conclusion

La résolution de l'énigme des pièces hexagonales sur le piédestal est une étape importante pour progresser dans Resident Evil 4 Remake. Les joueurs doivent trouver les quatre pièces hexagonales dans différents endroits du jeu, puis les placer dans le bon ordre sur le piédestal pour ouvrir la porte et continuer leur aventure.
